# Pleroma Docker

## Build

	$ docker-compose build

## Setup
	$ cp .env.example .env
	$ docker-compose pull
	$ ./setup.sh init
	$ docker-compose up

## Get current config

	$ ./setup.sh get

## Put config

	$ ./setup.sh put

## Destroy

	$ ./setup.sh destroy
